# GhostLoader
GhostLoader is an automatic manager for EmulationStation (ArkOS). It monitors and backs up your theme settings, restores profiles when changed, and ensures smooth transitions between themes. Installation, management, and uninstallation via an interactive menu.

# Installation 
Install GhostLoader.sh in roms/tools or roms2/tools if 2 sd cards

# Update
Added two new functions: 
- Backup all profiles
- Restore all profiles
